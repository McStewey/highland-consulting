# Highland Consulting

A bespoke, responsive multi-page website for Highland Consulting.

## Pages

- `index.html` — Capabilities and approach
- `about.html` — Firm point of view and leadership
- `insights.html` — Field Notes editorial hub
- `contact.html` — Contact and inquiry experience

The site is intentionally dependency-free and optimized for GitHub Pages. Open `index.html` locally or serve the directory with any static server.
